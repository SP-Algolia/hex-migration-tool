Hex Migration Tool

Transform your Hex queries for seamless Databricks compatibility with intelligent schema mapping.

How to Use This Tool

1

Export from Hex

In your Hex project, go to Project Settings ‚Üí Export ‚Üí Download as YAML. This downloads your project configuration with all queries and cells.

2

Upload YAML File

Drag and drop your exported .yaml file to the upload area below, or click to browse and select it from your computer.

3

Automatic Migration

The tool automatically converts Redshift functions to Databricks equivalents, updates table references, and optimizes queries for better performance.

4

Download & Import

Download the migrated YAML file and import it into your Databricks-connected Hex environment. Your queries are now ready for Databricks!
